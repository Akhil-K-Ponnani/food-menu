# Food Menu
***
Food Menu is a MERN Stack Application.

## Technologies Used
***
* Nodejs
* Express
* React
* MongoDB
* Bootstrap

## Installation
***
A little intro about the installation. 
```
$ git clone https://github.com/Akhil-K-Ponnani/food-menu

**To setup Frontend**
$ cd client
$ npm install
$ npm start

**To setup Backend**
$ cd server
$ npm install
$ npm start
